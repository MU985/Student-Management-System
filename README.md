# Student Management System (C++)

##  Overview
This is a simple console-based Student Management System written in C++.  
The program allows users to store and manage student records efficiently using a menu-driven interface.

It demonstrates core C++ programming concepts such as structures, functions, loops, and arrays.

---

##  Features

 Add student details  
 Display all student records  
 Search student by roll number  
 Delete student record  
 Menu-driven interface
